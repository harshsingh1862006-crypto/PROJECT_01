# User Authentication System – Prodigy InfoTech Internship Task 01

This is a simple full-stack authentication system project Based on the Next.js framework. HTML, Tailwind CSS, Typescript, React and PostgreSQL with Prisma as API have been used to make the project.
The project was made as Task 01 for the Prodigy InfoTech Full Stack Web Development Internship.

---

# Features

- User Registration
- User Login
- User Logout
- Password Encryption using bcryptjs
- Session-based Authentication
- Protected Dashboard Page
- PostgreSQL Database Connection
- Prisma API
- Simple and Responsive UI

---

## Frontend

# How to use

## Step 1

open the terminal and run

npm install

## Step 2
Download and install PostgreSQL and pgAdmin4

## Step 3
Run: 
```bash
npm run db:generate
npm run db:push
```
this generates the Database and needed tables

## Step 4
This is the final step to Build and Start the website on http://localhost:3000/
Run: 
```bash
npm run build
npm run start
```
